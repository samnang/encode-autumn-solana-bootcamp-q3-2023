# Example: Hello Anchor

## Install node dependencies

```bash
❯ yarn install

```

## Build Solana program

```bash
❯ anchor build

```

## Run tests

```bash
❯ anchor test

```

## Start local node

```bash
❯ yarn run local-node

```

## Deploy Solana program

```bash
❯ anchor deploy

```

## Run client to interact with Solana program

```bash
❯ yarn run client

```
